# trinkelle by amna
 

Overview
Trinkelle by Amna is a simple and aesthetic frontend website for a handmade beaded jewelry store.  
It is designed using only HTML, CSS, and JavaScript in a single file, making it easy to run and showcase.

The goal of this project is to create a soft, elegant, and visually appealing online store interface without any backend complexity.

Features
- Soft pastel aesthetic design 🎀  
- Responsive product grid layout 💍  
- Smooth hover animations ✨  
- Simple interactive buttons (frontend only)  
- Clean and minimal UI design  
- Fully single-file project (easy to run)

---

Purpose of the Project
This project was created to practice frontend development skills and improve understanding of:
- HTML structure
- CSS styling and animations
- Basic JavaScript interactions
- UI/UX design principles

It also demonstrates how a simple idea can be turned into a visually appealing website.
 
Technologies Used
- HTML5  
- CSS3  
- JavaScript (Vanilla)

---

 
 Design Style
The design is inspired by:
- Soft pastel colors  
- Minimal aesthetic layouts
